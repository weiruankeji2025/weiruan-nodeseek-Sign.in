# NodeSeek 增强助手

简洁高效的 NodeSeek 论坛增强油猴脚本。

## 功能特点

| 功能 | 说明 |
|------|------|
| **自动签到** | 每日自动签到，支持随机/固定鸡腿模式 |
| **关键字监控** | 自定义关键字实时监控，支持精准/模糊匹配 |
| **消息弹窗** | 关键字匹配时弹窗提醒，点击跳转帖子 |
| **自动翻页** | 自动翻页浏览，可自定义翻页间隔 |
| **最新交易** | 侧边栏显示5个正在进行的交易 |
| **最新抽奖** | 侧边栏显示5个正在进行的抽奖帖 |
| **骗子曝光** | 全站索引骗子相关帖子 |
| **已浏览标记** | 点击过的帖子显示"已看"标记 |
| **中奖提醒** | 自动检测参与的抽奖，中奖时弹窗通知 |
| **深色模式** | 自动适配系统主题 |

## 安装方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击安装脚本：[nodeseek-auto-checkin.user.js](./nodeseek-auto-checkin.user.js)
3. 访问 NodeSeek 论坛即可自动生效

## 配置选项

在 Tampermonkey 中编辑脚本，找到 `CONFIG` 对象进行配置：

### 基础配置

```javascript
const CONFIG = {
    RANDOM_MODE: true,     // 签到模式：true=随机鸡腿, false=固定5鸡腿
    TRADE_COUNT: 5,        // 交易显示数量
    LOTTERY_COUNT: 5,      // 抽奖显示数量
    SCAM_COUNT: 5,         // 骗子曝光显示数量
    WIN_CHECK_INTERVAL: 10 * 60 * 1000  // 中奖检测间隔（毫秒）
};
```

### 关键字监控配置

```javascript
// 精准匹配关键字（完全匹配标题中的词）
KEYWORDS_EXACT: ['VPS', 'CN2', 'GIA'],

// 模糊匹配关键字（标题包含即匹配）
KEYWORDS_FUZZY: ['优惠', '特价', '免费', '白嫖', '羊毛'],

// 监控间隔（毫秒），默认30秒
KEYWORD_MONITOR_INTERVAL: 30 * 1000,

// 是否启用关键字监控
KEYWORD_MONITOR_ENABLED: true,
```

**关键字匹配模式说明：**

| 模式 | 配置项 | 说明 |
|------|--------|------|
| **精准匹配** | `KEYWORDS_EXACT` | 标题中必须包含完整关键词 |
| **模糊匹配** | `KEYWORDS_FUZZY` | 标题包含关键词即匹配 |

### 自动翻页配置

```javascript
// 是否启用自动翻页（默认关闭）
AUTO_PAGE_ENABLED: false,

// 翻页间隔（秒），默认60秒
AUTO_PAGE_INTERVAL: 60
```

**自动翻页使用说明：**

1. 在侧边栏点击「启动」按钮开始自动翻页
2. 倒计时显示距离下次翻页的时间
3. 点击「停止」暂停自动翻页
4. 点击「下一页」手动翻页

### 签到模式说明

| 模式 | 配置值 | 说明 |
|------|--------|------|
| **随机模式** | `RANDOM_MODE: true` | 随机获得0-10个鸡腿（默认） |
| **固定模式** | `RANDOM_MODE: false` | 固定获得5个鸡腿 |
